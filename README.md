# Syncer

File synchronization tool

## Command line options

```
--src (required)
    source sync dir
--dst (required)
    destination sync dir

--interval (default: '30s')
    full sync interval

--force (default: 'false')
    force mode (ignore lockfile)

--log (disabled by default)
    log file
--format (default: 'json')
    output log format [text, json]
--level (default: 'debug')
    log level
```
